# Sales & Cost Dashboard Project

## Overview
This project analyzes over years of sales and cost data using Excel and Power BI to monitor business performance, profitability, and sales trends.

## Goal
Developed a comprehensive Power BI reporting solution that transforms sales and cost data into actionable insights, enabling performance monitoring, trend analysis, and informed business decisions.

## Dashboard Highlights
Sales Dashboard: Developed an interactive Power BI dashboard to analyze sales performance, revenue trends, and KPIs using dynamic filters and visual reports

Comparison Dashboard: Enabled period-over-period sales analysis through dynamic current and previous date selections, measuring performance changes across revenue, delivery apps, average sales, and top product categories.

Cost Analysis Dashboard: Tracked direct and indirect costs, supplier expenses, and profitability indicators to support cost control and margin improvement.
