# Sales & Cost Dashboard Project

## Overview
This project analyzes sales and cost data using Excel and Power BI.

## Tools
- Power BI
- Excel

## Goal
Developed a comprehensive Power BI reporting solution that transforms sales and cost data into actionable insights, enabling performance monitoring, trend analysis, and informed business decisions.

## Key Insights
Developed an interactive Power BI dashboard to analyze sales performance, revenue trends, and KPIs using dynamic filters and visual reports

Comparison Dashboard: Enabled period-over-period sales analysis through dynamic current and previous date selections, measuring performance changes across revenue, delivery apps, average sales, and top product categories.

Cost Analysis Dashboard: Tracked costs, supplier expenses, and profitability metrics
