# Sales & Cost Dashboard Project

## Overview
This project analyzes sales and cost data using Power BI.

## Tools
- Power BI
- Excel

## Key Insights
1- Developed an interactive Power BI dashboard for “Caf´e Khalih ‘Aziz (Az
   Bakery)” to monitor daily cash sales, product sales, and cost tracking.
2- Designed a multi-sheet dashboard with dynamic date filters to compare sales
   performance across different periods and identify peak hours and best-selling
   items.
3- Created dedicated sheets for detailed sales breakdowns by product, category, and
   transaction level, enabling accurate revenue analysis per menu item.
4- Built a cost-tracking sheet to monitor direct and indirect costs and track supplier
   expenses, helping the owner control margins and reduce unnecessary spending.
5- Delivered a user-friendly reporting solution that improved management
   decision-making by transforming raw sales and cost data into clear visual
   dashboards and KPIs.
